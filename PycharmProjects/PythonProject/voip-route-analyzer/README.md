# 📞 VoIP Route Analyzer Lite

A Python + Streamlit dashboard that simulates NOC (Network Operations Center) workflows for VoIP route quality analysis.

🔗 Live Demo: (https://voip-route-analyser-lite-ria.streamlit.app)  
💻 Source Code: https://github.com/AlexandraPoe/voip-route-analyser-lite

## 🚀 Features

- ASR (Answer-Seizure Ratio) calculation
- PDD (Post Dial Delay) analysis
- SIP failure pattern detection
- Missing RBT detection
- FAS (Fake Answer Supervision) fraud suspicion
- Multi-route comparison and ranking

## 🧠 Why this project

This project simulates real-world telecom scenarios such as:

- Healthy routing
- Carrier degradation
- Fraud patterns (FAS)
- Poor user experience (missing RBT)

It demonstrates practical NOC thinking, not just coding.
